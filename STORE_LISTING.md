## Detailed Description

AlertLens AI is the best AI-powered Kubernetes alert summarizer Chrome extension for SREs, DevOps engineers, and platform teams. Instantly analyze K8s alerts, generate runbooks, correlate incidents, and get actionable remediation steps — right from PagerDuty, Grafana, or Datadog.

### Key Features

- **AI Alert Summarization** — Paste any Kubernetes alert (JSON or text) and get a clear, actionable summary in seconds
- **Severity Classification** — Automatic P0-P4 severity assessment with color-coded badges
- **Runbook Generation** — AI-generated step-by-step remediation runbooks for each alert
- **Incident Correlation** — Identify related alerts and potential root causes across your cluster
- **Slack-Ready Summaries** — One-click copy of formatted alert summaries for Slack or Teams
- **PagerDuty / Grafana / Datadog Integration** — Content script works directly on monitoring platform pages
- **Side Panel Workspace** — Full analysis workspace with tabs for different analysis modes
- **Dark & Light Themes** — Designed for on-call engineers working at any hour

### How It Works

1. Click the AlertLens AI icon or open it from PagerDuty/Grafana/Datadog
2. Paste your Kubernetes alert JSON or text
3. Click "Analyze Alert" for instant AI-powered analysis
4. Review the severity, summary, root cause, and remediation steps
5. Copy the Slack-formatted summary for your incident channel

### Use Cases

- SREs triaging Kubernetes alerts during on-call shifts
- DevOps engineers investigating pod crashes and resource issues
- Platform teams correlating alerts across namespaces and clusters
- Incident commanders needing quick alert summaries for stakeholders
- Teams using PagerDuty, Grafana, or Datadog for Kubernetes monitoring

### Technical Details

- Manifest V3 Chrome extension (latest standard)
- Integrates with PagerDuty, Grafana, and Datadog pages
- AI-powered streaming analysis
- Supports JSON and plain-text alert formats
- Lightweight with minimal permissions

### Privacy

AlertLens AI sends alert text to our AI API for analysis only. Alert data is not stored permanently. No monitoring platform credentials are accessed.
